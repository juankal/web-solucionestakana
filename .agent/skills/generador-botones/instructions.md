Cuando el usuario te pida crear un botón, debes seguir estas reglas:
1. Usa siempre clases de utilidad modernas de CSS.
2. Asegúrate de incluir estados interactivos como hover, focus y active.
3. El botón debe cumplir con estándares de accesibilidad (atributos ARIA).
4. Genera una versión con bordes redondeados y otra con bordes rectos.
5. Ejecuta la herramienta de validación asociada antes de entregar el código.
